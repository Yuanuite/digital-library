{{/* 层级感知 prev/next：封面→章节间→完结 */}} {{- $pages := slice -}} {{- $parent := .Parent -}} {{- $isSection := .IsSection -}} {{- /* 获取同级页面列表 */ -}} {{- if $parent -}} {{- if $isSection -}} {{- /* 书封面：在 books/ 下，同级是其他书 */ -}} {{- $pages = $parent.Pages.ByWeight -}} {{- else -}} {{- /* 章节页：同级是同一本书的其他章节 */ -}} {{- $pages = $parent.RegularPages.ByWeight -}} {{- if not $pages -}} {{- $pages = $parent.Pages.ByWeight -}} {{- end -}} {{- end -}} {{- end -}} {{- if gt (len $pages) 1 -}} {{- $prev := false -}}{{- $next := false -}}{{- $found := false -}}{{- $isLast := false -}} {{- $lastIdx := sub (len $pages) 1 -}} {{- range $idx, $p := $pages -}} {{- if $found -}} {{- if not $next -}}{{- $next = $p -}}{{- end -}} {{- end -}} {{- if eq $p $ -}} {{- $found = true -}} {{- if eq $idx $lastIdx -}}{{- $isLast = true -}}{{- end -}} {{- else -}} {{- $prev = $p -}} {{- end -}} {{- end -}} {{- /* 特殊处理：书封面页的 prev 指向书架 */ -}} {{- if and $isSection (not $prev) -}} {{- $booksSection := .Site.GetPage "/books" -}} {{- if $booksSection -}}{{- $prev = $booksSection -}}{{- end -}} {{- end -}}
{{- with $prev -}} ← {{ partial "docs/title" . }} {{- end -}}
{{- if $isLast -}} {{- /* 最后一页：完结 */ -}} 完结 · 返回书籍界面 → {{- else if $next -}} {{ partial "docs/title" $next }} → {{- end -}}
{{- end -}}
