{{/* 层级感知 prev/next */}} {{- $pages := slice -}} {{- $parent := .Parent -}} {{- $isSection := .IsSection -}} {{- if $parent -}} {{- if $isSection -}} {{- /* 书封面：同级是其他书 */ -}} {{- $pages = $parent.Pages.ByWeight -}} {{- else -}} {{- /* 章节页：同级是同一本书的其他章节 */ -}} {{- $pages = $parent.RegularPages.ByWeight -}} {{- if not $pages -}} {{- $pages = $parent.Pages.ByWeight -}} {{- end -}} {{- end -}} {{- end -}} {{- $prev := false -}}{{- $next := false -}}{{- $nextLabel := "→" -}}{{- $isLast := false -}} {{- if $isSection -}} {{- /* 书封面：prev=分类页(优先) 否则书架，next=开始阅读 */ -}} {{- $prev = .Site.GetPage "/books" -}} {{- with .Params.categories -}} {{- $cat := index . 0 -}} {{- range $.Site.Taxonomies.categories -}} {{- if eq .Page.Title $cat -}} {{- $prev = .Page -}} {{- end -}} {{- end -}} {{- end -}} {{- with .RegularPages.ByWeight -}} {{- if gt (len .) 0 -}}{{- $next = index . 0 -}}{{- $nextLabel = "开始阅读 →" -}}{{- end -}} {{- end -}} {{- else if gt (len $pages) 1 -}} {{- $found := false -}}{{- $isLast := false -}} {{- $lastIdx := sub (len $pages) 1 -}} {{- range $idx, $p := $pages -}} {{- if $found -}} {{- if not $next -}}{{- $next = $p -}}{{- end -}} {{- end -}} {{- if eq $p $ -}} {{- $found = true -}} {{- if eq $idx $lastIdx -}}{{- $isLast = true -}}{{- end -}} {{- else -}} {{- $prev = $p -}} {{- end -}} {{- end -}} {{- if $isLast -}} {{- $next = $parent -}}{{- $nextLabel = "完结 · 返回书籍界面 →" -}} {{- end -}} {{- end -}} {{- if or $prev $next -}}
{{- with $prev -}} ← {{ partial "docs/title" . }} {{- end -}}
{{- with $next -}} {{ $nextLabel }} {{- end -}}
{{- end -}}
