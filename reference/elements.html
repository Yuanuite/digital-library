    定义马尔可夫决策过程（MDP）
    一个 MDP 由状态集 $S$、动作集 $A$、转移概率 $P(s'|s,a)$、奖励函数 $R$ 和折扣因子 $\gamma$ 组成。其核心性质是马尔可夫性：未来只取决于当前状态和动作，与历史无关。

{{< /theorem >}}
    定理贝尔曼最优方程

    最优状态价值函数满足：

    $$ V^*(s) = \max_a \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^*(s') \right] $$

    例题例1-1 远期合约收益

    假设今天是1月15日，某加工商知道在5月15日需要100000磅铜。即期铜价为每磅340美分，5月份期货合约为每磅320美分。该加工商应如何对冲？

    进入4份期货合约多头（每份25000磅），锁定价格在每磅320美分。整体费用约 $320000$ 美元，无论5月即期价格如何变动。

\begin{algorithm}
\caption{算法 1: Sarsa}
\begin{algorithmic}
state 输入: episodes, α, γ
for{each episode}
    state S ← first state
    repeat
        state Q(S, A) ← Q(S, A) + α(R + γQ(S', A') - Q(S, A))
    until{S is terminal state}
endfor
\end{algorithmic}
\end{algorithm}

    算法Sarsa 算法
    \begin{algorithm}
\caption{算法 1: Sarsa}
\begin{algorithmic}
\REQUIRE episodes, $\alpha$, $\gamma$
\FOR{each episode}
    \STATE $S \leftarrow$ first state
    \REPEAT
        \STATE $Q(S, A) \leftarrow Q(S, A) + \alpha(R + \gamma Q(S', A') - Q(S, A))$
    \UNTIL{$S$ is terminal state}
\ENDFOR
\end{algorithmic}
\end{algorithm}

    代码块#

    标 python（或其他语言）才有语法高亮：

    ```python
def value_iteration(V, theta=1e-6):
    while True:
        delta = 0
        for s in states:
            v = V[s]
            V[s] = max(q_value(s, a) for a in actions)
            delta = max(delta, abs(v - V[s]))
        if delta < theta:
            break
    return V
```
