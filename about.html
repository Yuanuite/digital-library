    关于本站#

    个人数字图书馆，基于 MkDocs Material 构建，部署于 GitHub Pages。

    阅读增强#

    本站启用了两项阅读辅助功能，在所有页面自动生效：

    术语悬浮提示#

    专业术语缩写（如 OCR、SVM、API）带有下划虚线，鼠标悬停即显示全称和解释。无需额外标记——只要在 snippets/abbreviations.md 中定义过，全站自动识别。

    试试：OCR、API、GDP、SVM、GARCH

    文本高亮#

    使用 ==文本== 语法标记重点内容，渲染为黄色高亮背景。

    在任意书籍 markdown 文件中使用 ==这段话很重要== 即可渲染为 ==这段话很重要== 的效果。

    技术栈#

    MkDocs Material · MathJax 3 · pseudocode.js · GitHub Pages
