    Analysis

    Spectra of Random Graphs with Given Expected Degrees — 论文解读#

    Fan Chung, Linyuan Lu, Van Vu — PNAS (2003)

    一、论文讲了什么故事#

    背景冲突#

    在论文写作的 2002-2003 年，关于**幂律图（power-law graph）**的谱分布存在两派互斥的观点：

    流派主张代表工作
    Wigner 半圆律派图的特征值应服从半圆分布Füredi-Komlós (1981) 对 Erdős–Rényi 随机图证明了这一点
    幂律分布派特征值自身也是幂律分布Faloutsos et al. (1999) 画了 Internet 拓扑邻接矩阵的特征值，明显不是半圆；Mihail-Papadimitriou 证明了 β > 3 时邻接矩阵特征值服从幂律

    核心矛盾：半圆律和幂律分布是两种完全不同的分布形状，谁都觉得自己是对的。到底谁对？

    核心洞察#

    Chung、Lu、Vu 的洞察是：双方都对，关键看你选哪个矩阵。同一个图可以关联三种不同矩阵：

    1. 邻接矩阵 A — 对最大度敏感（局部性质）
    2. 组合 Laplacian D − A — 计数生成树
    3. 归一化 Laplacian L = I − D⁻¹/²AD⁻¹/² — 控制图的展开性/随机游走混合速率（全局性质）

    在经典的 Erdős–Rényi 随机图中，所有顶点的度几乎相等，三种矩阵的谱本质上等价（只差一个伸缩或平移）。但幂律图度分布极不均匀，三种矩阵的行为可以完全不同。

    核心结论#

    邻接矩阵的谱 → 受最大度支配，服从幂律分布
    归一化 Laplacian 的谱 → 服从 Wigner 半圆律

    一条线把两个看似矛盾的学派统一了。

    二、主要定理#

    随机图模型 G(w)#

    给定期望度序列 w = (w₁, ..., wₙ)，每条边 (i,j) 以概率 pᵢⱼ = wᵢwⱼρ 独立出现，其中 ρ = 1 / Σwᵢ。

    两个关键参数：

    • d̃ = Σwᵢ² / Σwᵢ — 二阶平均度（加权平方和的平均）
    • m = maxᵢ wᵢ — 最大期望度

    邻接矩阵最大特征值（定理 1–3）#

    ||A|| ≈ max{√m, d̃}

    • 若 d̃ > √m·log n → ||A|| ≈ d̃（定理 1）
    • 若 √m > d̃·log² n → ||A|| ≈ √m（定理 2）
    • 第 k 大特征值 ≈ √mₖ，其中 mₖ 是第 k 大期望度

    幂律图相变（定理 4）#

    对于指数为 β 的幂律图，有一个关键的相变点 β = 2.5：

    β 范围最大特征值行为
    β ≥ 3(1+o(1))√m
    2.5 < β < 3(1+o(1))√m
    2 < β < 2.5(1+o(1))d̃ ∝ c·m³⁻ᵝ

    且对 β > 2.5，前 k 大特征值服从指数为 2β − 1 的幂律分布（这解释了为什么 Internet 图的邻接矩阵特征值看起来是幂律——因为 Internet 的 β ≈ 2.1 ∼ 2.4）。

    Laplacian 半圆律（定理 5–6）#

    在 wₘᵢₙ ≫ √w̄ 条件下：

    ||C|| = (1+o(1))·2/√w̄

    且 Cₙₒᵣ 的特征值分布收敛到 Wigner 半圆律，密度函数 w(x) = (2/π)√(1−x²)。

    三、具体证明过程#

    邻接矩阵部分的证明策略#

    邻接矩阵最大特征值的证明主要用矩方法 + 组合计数：

    1. 下界：容易验证 ||A|| ≥ max{d̃, √m}（用 Rayleigh 商：d̃ 来自全 1 向量，√m 来自最大度顶点对应的单位向量）

    2. 上界：证明 ||A|| ≤ 7√(log n) · max{√m, d̃}（定理 3），核心工具是高阶矩迹方法——||A||²ᵏ ≤ Trace(A²ᵏ)，而 Trace(A²ᵏ) 展开为图上所有长为 2k 的闭游走之和，每项的期望可以按游走经过的不同边数分类，利用 Füredi-Komlós 的组合引理控制各类游走的数量。

    3. 精度提升：在 √m 和 d̃ 相差 log n 量级以上时，去掉 √(log n) 因子得到渐近精确结果（定理 1、2）。

    Laplacian 半圆律的证明（核心难点）#

    这里用到了论文中最精巧的技巧。定义：

    M = D⁻¹/²AD⁻¹/² − P₀

    C = W⁻¹/²AW⁻¹/² − χ*χ

    其中 C 是 M 的"期望版本"——用期望度 wᵢ 替换实际度 dᵢ。证明分五步：

    第一步：Wigner 高阶矩方法#

    对于任意正整数 k：

    E(λ₁(C)²ᵏ) ≤ E(Trace(C²ᵏ))

    Trace(C²ᵏ) 展开为 Kₙ 上所有长为 2k 的闭游走的贡献之和。cᵢⱼ 是零均值独立随机变量（在 W⁻¹/²AW⁻¹/² − χ*χ 这种中心化形式下），因此只当游走中每条边出现至少 2 次时（称为"好游走"），该项期望才非零。

    第二步：边的贡献界#

    对每条边 (i,j)，计算其 m 次幂的期望：

    |E(cᵢⱼᵐ)| ≤ ρ / (wₘᵢₙ)^(m−2)

    这里用了一个精妙的恒等式：cᵢⱼ = qᵢⱼ/√(wᵢwⱼ) 以概率 pᵢⱼ，cᵢⱼ = −pᵢⱼ/√(wᵢwⱼ) 以概率 qᵢⱼ = 1−pᵢⱼ。

    第三步：Füredi-Komlós 组合引理#

    游走计数引理：在 Kₙ 上经过 l+1 个不同顶点、长为 2k 的闭好游走数量满足：

    |Wₗ,ₖ| ≤ n(n−1)⋯(n−l) · C(2k,2l) · C(2l,l) · (1/(l+1)) · (l+1)⁴⁽ᵏ⁻ˡ⁾

    第四步：求和与控制#

    将贡献代入：

    E(Trace(C²ᵏ)) ≤ Σₗ₌₀ᵏ |Wₗ,ₖ| · ρˡ / wₘᵢₙ²ᵏ⁻²ˡ

    取 k = g(n)log n（g(n) 缓慢增长），在 wₘᵢₙ ≫ √w̄·log³ n 条件下，求和被末项 l=k 支配（相邻项之比 ≫ 1）：

    E(Trace(C²ᵏ)) ≤ (1+o(1)) ρᵏ nᵏ⁺¹ 4ᵏ

    得到：

    E(λ₁(C)²ᵏ) ≤ (1+o(1)) n (2/√w̄)²ᵏ

    用 Markov 不等式，由于 k = Ω(log n)，可取 ε(n) → 0 使得 (1+ε)⁻²ᵏ 以超多项式速度衰减，证得 ||C|| ≤ (1+o(1))·2/√w̄。

    第五步：矩收敛 → 半圆律（定理 6）#

    为证特征值经验分布收敛到半圆律，需证对任意固定 s，Cₙₒᵣ 的 s 阶矩收敛到半圆分布的 s 阶矩。半圆律的偶数阶矩为 Catalan 数 (2k)!/2²ᵏk!(k+1)!，奇数阶矩为 0。

    • 偶数阶：回到 Füredi-Komlós 引理的精确形式，当 l=k 时上界恰为等式，得 E(Trace(C²ᵏ)) = (1+o(1)) s'ₖ,ₖ，归一化后精确得到 Catalan 数。
    • 奇数阶：奇长度游走必有某边恰好出现 1 次，期望为 0。需要考虑的游走最多用 k 条不同边，但在 wₘᵢₙ ≫ √w̄ 下，归一化后的贡献 → 0。

    这完成了半圆律的证明。

    四、论文的方法论贡献#

    1. 统一框架：G(w) 模型以任意期望度序列为输入，Erdős–Rényi 只是等权重的特例，幂律图是对应 wᵢ ∝ i⁻¹/⁽ᵝ⁻¹⁾ 的特例。

    2. 相变发现：β = 2.5 是幂律图邻接矩阵谱行为的相变点——当 β < 2.5 时，最大特征值从 √m 切换到 d̃ ∝ m³⁻ᵝ，这个值远小于 √m。

    3. 矩阵选择的重要性：同一张图用不同矩阵（邻接矩阵 vs Laplacian）得到的谱分布可以截然不同——这对后来网络科学中选择合适的矩阵表示有深远影响。

    这篇论文用极其精炼的 8 页篇幅解决了随机图谱理论中的一个核心争论，是谱图论和网络科学的必读文献。
