# 关于本站

个人数字图书馆，基于 [Hugo](https://gohugo.io) + [Hugo Book](https://github.com/alex-shpak/hugo-book) 主题构建，部署于 GitHub Pages。

## 技术栈

[Hugo](https://gohugo.io) · [KaTeX](https://katex.org) · [pseudocode.js](https://github.com/SaswatPadhi/pseudocode.js) · [GitHub Pages](https://pages.github.com)
